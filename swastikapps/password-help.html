Help-Password

Introduction

Welcome to PASSWORD. PASSWORD is a Password Manager App. You can save all of your passwords here. App interface is very simple. App also uses Encryption for secure your data.


Add Record

STEP 1:
You can add new record by following ways:
(1) Click on 'ADD' floating button at Bottom right of screen.
OR (2) Click on 'Add' option in Navigation Drawer.

STEP 2:
Select category of Record from drop-down list.Based on category, text fields will appear.
Enter all fields and click on Add button.
Your record will be saved in database.


Modify Record

STEP 1:
Click on Top right 3-dot button on Record.
A menu will open. Click on Modify option.

STEP 2:
If you want to change category of record, select it from drop-down list. If not, keep it as it is.

STEP 3:
Change fields whatever you want.
Click on Modify button.
Your record will be modified.


Information of Record

STEP 1:
Simply click on record. All informations about it will be shown.
You can also choose Info option from menu of record.


Mark as Favourite

STEP 1:
Click on STAR icon on Record.
If record is not favourite, it will be set as favourite.
Star icon color will change from black to yellow.

STEP 2:
To Unfavourite, click on Star icon.
Record will be Unfavourite and Star color will change from yellow to black.


Delete Record

STEP 1:
Choose Delete option from menu of record.
A Confirm dialog box will appear. Click OK to delete record.


Show Password

STEP 1:
Click on EYE icon on record.
It will show password.

STEP 2:
To hide it, click again on EYE icon. It will hide password and show only stars.
You can see password also in Info option.

STEP 3:
If you don't want password and EYE icon to be shown on Record, disable it in Setting.


Where is my Record?

If your record is Favourite, It will be on App Home screen.
If not, Select All Records from Navigation Drawer. You will file required record.
You can find record easily based on its category.
e.g., If record category is Email, Select Email from Navigation Drawer. You will find records within Email category.


Backup

STEP 1:
Select Backup option from Navigation Drawer.

STEP 2:
Select location where to save Backup file.

STEP 3:
If you want to set password for backup file, check the checkbox.
A password field will appear. Insert your password there.

STEP 4:
Enter file name whatever you want. At last click on Backup button. Your file will be saved to your device. This file will have '.spbf' extension.
(NOTE: If you set password for Backup file, You must have to remember it. It must be required at Restore time.)


Restore

STEP 1:
Select Restore option from Navigation Drawer.

STEP 2:
Select location where you have saved your Backup file.

STEP 3:
Click on 'Select File' button.
A dialog box will appear. It will have list of all files in selected folder.
Select a file which you want to restore.

STEP 4:
If selected file is password protected, It will ask for password. Type password of that file.
STEP 5:
Click Restore button.
All records from backup file will be restored.


Setting

You can customize this app with Setting.
In Setting, you will have following options:

'Theme':
You can set theme for this app. Currently there are two themes:(1)Light (2)Dark.

'Animation':
You can enable or disable use of Animations in this app.

'Records on Home screen':
You can set wether to display all records or only favourite records on Home screen.

'Don't show password':
If you enable it, it will hide password and EYE icon on record.

'Help':
you will get this Help page.


Reset Master Password

STEP 1:
Select Reset Password option from top left menu.
STEP 2:
You will be asked for old password. Enter it.
STEP 3:
Enter new password.

STEP 4:
Click on OK and your master password will be reset.


Info

Click on About in Navigation Drawer. You will get Information about this app and its version.


Support

You will get Rate and Share option from top right menu.
Thanks for using this app.
